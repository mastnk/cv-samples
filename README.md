# Machine Learning Examples for Computer Vision

A collection of machine learning examples and reference implementations for computer vision.

## Getting Started

1. Click an "Open in Colab" button.
2. Open the notebook in Google Colab.
3. (Option) Select Runtime → Change runtime type → GPU.
4. Run all cells.
5. Upload your own image if required.


## [YOLO (You Only Look Once)](https://github.com/ultralytics/ultralytics)

Assigns a single label to the entire image from 1000 ImageNet categories.  
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg) Classification](https://colab.research.google.com/github/mastnk/cv-samples/blob/main/yolo/yolo_classification.ipynb)

Detects multiple objects in an image and draws bounding boxes with confidence scores.  
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg) Detection](https://colab.research.google.com/github/mastnk/cv-samples/blob/main/yolo/yolo_detection.ipynb)

Detects objects and draws pixel-level masks to show the exact shape of each object.  
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg) Segmentation](https://colab.research.google.com/github/mastnk/cv-samples/blob/main/yolo/yolo_segmentation.ipynb)

Detects people and estimates body keypoints such as joints and facial landmarks.  
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg) Pose](https://colab.research.google.com/github/mastnk/cv-samples/blob/main/yolo/yolo_pose.ipynb)
